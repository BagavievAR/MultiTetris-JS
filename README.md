# MultiTetris-JS
MultiTetris JS — это веб-приложение с реализацией классической игры «Тетрис», поддерживающее:

одиночный режим (Solo),

многопользовательские режимы (Score Battle, Battle Royale),

авторизацию пользователей,

сохранение результатов и лидерборды.

Проект реализован в виде monorepo и состоит из клиентской и серверной частей.

## Установка зависимостей

Все зависимости устанавливаются одной командой из корня проекта:
npm install
Используются npm workspaces, поэтому отдельно заходить в client или server не требуется.

# Запуск проекта
1. Запуск сервера

Из корня проекта:

npm run dev:server

Сервер будет доступен по адресу:

http://localhost:3000

При первом запуске автоматически создаётся файл базы данных SQLite:

multitetris.db

2. Запуск клиента

В новом окне терминала, из корня проекта:

npm run dev:client

Клиент будет доступен по адресу:

http://localhost:5173

## Авторизация

В приложении реализована система регистрации и входа:

данные пользователей хранятся в SQLite,

используется JWT-аутентификация,

без авторизации можно играть как гость (результаты не сохраняются).

## Игровые режимы
### Solo

классический одиночный Тетрис,

сохранение результатов в профиль пользователя,

история игр и лидерборд.

### Score Battle

соревновательный режим,

создание и подключение к комнатам,

лобби с готовностью игроков.

### Battle Royale

режим на нескольких игроков,

выбывание при поражении,

комнаты с кодом приглашения.
